This project is a Python-based web scraper designed to extract structured information from online articles.  
The scraper uses **Requests**, **BeautifulSoup**, **Pandas**, and **Flask** to:

- Fetch article content from the web  
- Parse and extract specific data (tables, text, lists, etc.)  
- Store extracted data in **Excel files**  
- Expose the results through a **REST API endpoint**

This repository was created as part of the *Python Web Scraping Series*.

---

##  Features

-  Scrape any publicly accessible webpage  
-  Extract data such as:
  - Headings
  - Tables
  - Lists
  - Paragraphs
-  Cleans and structures extracted content  
-  Saves output to Excel (`.xlsx`)  
-  Provides a Flask API endpoint for real-time scraping  
-  Includes user-agent headers to bypass blocking  
-  Easy to extend for custom scraping tasks
